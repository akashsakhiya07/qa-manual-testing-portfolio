## Bug 1: Add to Cart button response delay on product page

Bug ID: BUG_001  
Module: Product / Cart  
Environment: Web Browser (Chrome)  

Severity: Medium  
Priority: Low  
Status: New  

Steps to Reproduce:
1. Open the application homepage  
2. Search for a product (e.g. MacBook)  
3. Open the product details page  
4. Click on the "Add to Cart" button  

Expected Result:
Product should be added to the cart immediately after clicking the button.

Actual Result:
Product is added to the cart after a noticeable delay of a few seconds.

## Bug 2: Error message alignment issue on login page

Bug ID: BUG_002  
Module: Login  
Environment: Web Browser  

Severity: Low  
Priority: Low  
Status: New  

Steps to Reproduce:
1. Open Login page  
2. Enter invalid email or password  
3. Click on Login button  

Expected Result:
Error message should be properly aligned and clearly visible.

Actual Result:
Error message appears slightly misaligned and overlaps with other text.
